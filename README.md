## React UI Component

### 使用方法

```js
  npm install mars-ui-pro
```

### 开发贡献

- 目录结构和代码规范
```
  |--.storybook storybook相关配置 
  |--node_modules
  |--public
  |--src
      |-- components
         |--Button
            |--button.tsx
            |--button.test.tsx
            |--style.scss 组件单独样式
      |-- styles 全局样式文件
        _variables.scss (各种变量以及可配置设置)
        _minxins.scss (全局mixins)
        _functions.scss (全局functions)
        index.scss (样式入口文件)
```
